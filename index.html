Eye Aspect Ratio (EAR)

  • Measures openness of eye
  • Uses 6 facial landmarks around eye
  • Mathematical formula:
    EAR = (|p₂-p₆| + |p₃-p₅|) / (2|p₁-p₄|)
  • Value decreases when eye closes
  • Threshold: EAR < 0.22 = closed eye

Blink to Morse Code Translation

  • Short blink (4 frames): Dot (.)
  • Long blink (16 frames): Dash (-)
  • Short pause (35 frames): Character separator (/)
  • Long pause (60 frames): Word separator (¦/)
  • Very long closure (100 frames): End session

Core Algorithm: Eye Aspect Ratio

def compute_eye_aspect_ratio(eye_points): # Calculate vertical distances vertical_dist_1 = math.dist(eye_points[1], eye_points[5]) vertical_dist_2 = math.dist(eye_points[2], eye_points[4]) # Calculate horizontal distance horizontal_dist = math.dist(eye_points[0], eye_points[3]) # Compute eye aspect ratio (EAR) ear = (vertical_dist_1 + vertical_dist_2) / (2.0 * horizontal_dist) return ear

Blink Detection Code

# Detect blinks and convert to Morse code if average_ear < EYE_AR_THRESH: blink_counter += 1 if blink_counter >= EYE_AR_CONSEC_FRAMES: eyes_closed = True else: if blink_counter >= EYE_AR_CONSEC_FRAMES_CLOSED: morse_word += "-" # Dash for long blink elif eyes_closed: morse_word += "." # Dot for short blink blink_counter = 0 eyes_closed = False

Key Parameters

Parameter Value Purpose
EYE_AR_THRESH 0.22 Threshold for detecting closed eyes
EYE_AR_CONSEC_FRAMES 4 Frames needed for a dot
EYE_AR_CONSEC_FRAMES_CLOSED 16 Frames needed for a dash
PAUSE_CONSEC_FRAMES 35 Frames to separate characters
WORD_PAUSE_CONSEC_FRAMES 60 Frames to separate words
BREAK_LOOP_FRAMES 100 Frames to end the session
